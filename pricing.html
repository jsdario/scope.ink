F.A.Q

Basically, Scope is a tool that ables you to measure the work of a developer. It uses an algorithm to determine how the work is being done into the code. With Scope, you can analyze several parts of the creative work as pull requests, commits, reviews done, tests, etc. and the software prints visualizations so it's easier to see what's happening around the code.

For the moment, Scope is only integrated with GitHub and GitLab, so you will have to use these services to stock your repositories. We are working hard and fast to integrate others services like Bitbucket and etc.

If you use GitHub or GitLab, you only need to have or to be collaborator of a repository. Insert the repos into Scope and the software will do the rest for you!

All data are stored and can be reviewed anytime. With Scope, we help you to visualize those data, making it easier to understand the workflow of your company. We understand that visualizing data it is much easier to see what's happening around! Additionally, Scope is a tool that motivates your team to work better and efficiently. Don't be lax or you will pay the next beer! ;)

As we said in our main page, Scope does not read any kind of data unless your e-mail. We do not use servers just only analysis client-side. Our mission is to create value for your company or project and we don't want to read or to sell your data. Anyways, you can check our Privacy Policy.

Altough Open Source project might use Scope for free, private projects need to pay for reading their data. You can try the software 14 days for free, and if you don't like Scope, you are free to stop using it. When 14 days have passed, you will have to pay for it if you want to keep using it.
